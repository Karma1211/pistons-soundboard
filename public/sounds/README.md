# Sounds Directory

Drop your MP3 files here. Each file name must match what's defined in `lib/sounds.ts`.

## Required Files

### HYPE
- `deetroit-basketball.mp3` — The famous PA call ("DEETROIT BASKETBALL!")
- `lets-go-pistons.mp3` — Arena chant
- `bad-boys.mp3` — "Bad Boys" by Inner Circle (the anthem)
- `player-intros.mp3` — LCA player intro sequence
- `final-countdown.mp3` — "The Final Countdown" by Europa
- `for-life.mp3` — Pistons rally track

### FX
- `buzzer.mp3` — End-of-game buzzer
- `air-horn.mp3` — Arena air horn
- `crowd-roar.mp3` — LCA crowd going crazy
- `cannon.mp3` — Pistons cannon shot
- `shot-clock.mp3` — 24-second shot clock buzz
- `crowd-cheer.mp3` — Big play crowd cheer

### LEGENDS
- `yes-sir.mp3` — Rip Hamilton "Yes Sir!"
- `rasheed.mp3` — Rasheed Wallace "Ball Don't Lie"
- `chauncey-billups.mp3` — PA call for Chauncey Billups
- `john-mason.mp3` — John Mason PA announcer classic

### ANTHEMS
- `pump-it-up-pistons.mp3` — "Pump It Up Pistons" (1990)
- `motor-city.mp3` — Motor City anthem
- `we-not-done.mp3` — Playoff hype track
